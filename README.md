# Binary Search Tree (BST)
Реализация двоичного дерева поиска на С++.
Проект поддерживает инвариант BST, ручное управление памятью и классические методы вставки, удаления, поиска.

## Возможности
- Вставка элемента с сохранением свойства BST
- Удаление узла
- Поиск минимума и максимума
- Поиск преемника и предшественника
- Обход дерева
- ASCII-визуализация дерева
- Безопасное освобождение памяти
- Явный запрет копирования и перемещения для преодтвращения ошибок владения

## Инвариант BST
Для каждого узла верно, что в левом поддереве все значения строго меньше value, а в правом поддереве больше или равны value;
Инвариант поддерживается при всех операциях вставки и удаления.

## Сложность операций
Вставка - O(h); (O(n) в худшем случае)
Удаление - O(h)
Поиск - O(h)
Обход дерева - O(n)
h - высота дерева.
Поскольку дерево не является сбалансированным, высота может достигать O(n)
